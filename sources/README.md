# מקורות (Sources)

כאן נאספים המקורות שעליהם נשען המחזמר: **תנ"ך + חז"ל** (מדרשים, תלמוד, פרשנים) ופרשנויות ספציפיות שהמשתמש מספק.

## איך להוסיף מקורות
המשתמש מנהל מקורות ב-**NotebookLM**, אך אין גישה ישירה אליו מהסביבה. דרכי העברה:
- **Google Drive** — להעלות קבצים; ל-Claude יש גישת קריאה ל-Drive.
- **הדבקה בצ'אט** — לקטעים קצרים.
- **קבצים ברפו** — להוסיף טקסט/PDF ישירות לתיקייה זו.

## כללי ברזל
- כל מובאה עם **מראה מקום מדויק** (ספר, פרק, פסוק; מסכת/דף; שם מדרש; שם פרשן).
- **אין להמציא** מדרשים או פסוקים. לא בטוח → לסמן `[לאמת]`.
