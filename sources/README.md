# מקורות (Sources)

כאן נאספים המקורות שעליהם נשען המחזמר: **תנ"ך + חז"ל** (מדרשים, תלמוד, פרשנים) ופרשנויות ספציפיות שהמשתמש מספק.

## איך מעבירים מקורות (שיטת העבודה — חסכונית בטוקנים)
המשתמש מנהל מקורות ב-**NotebookLM**, אך אין גישה ישירה אליו מהסביבה.
**הדרך המומלצת: Google Drive.** ל-Claude יש גישת קריאה ל-Drive, וכך אפשר לעבוד עם מסמכים גדולים בלי לנפח את הרפו או הצ'אט:

1. המשתמש מעלה ל-Drive את קובצי המקור (מומלץ תיקייה ייעודית, למשל `King David Sources`).
2. Claude קורא **לפי דרישה** רק את הקטע הרלוונטי לתחנה שעליה עובדים (`mcp__Google_Drive__search_files` + `read_file_content`).
3. Claude **מזקק** לתוך קבצי `sources/` רק את המובאות המדויקות (עם מראה מקום) — לא את כל הטקסט.

דרכים חלופיות: הדבקה בצ'אט (לקטעים קצרים) או הוספת קבצים ישירות לתיקייה זו.

## כללי ברזל
- כל מובאה עם **מראה מקום מדויק** (ספר, פרק, פסוק; מסכת/דף; שם מדרש; שם פרשן).
- **אין להמציא** מדרשים או פסוקים. לא בטוח → לסמן `[לאמת]`.

## קבצים ומבנה
- `tehillim-map.md` — מיפוי מזמור תהילים לכל תחנה בחיי דוד.
- `station-NN-name.md` — דף זיקוק מקורות לכל תחנה (לפי סדר הציר). תבנית קבועה:
  מקור עיקרי → פסוקי מפתח → תמצית → ניתוח דרמטי → חיבור לתהילים → מקבילה מודרנית → `[להעשיר]` מחז"ל/הפרשנויות.
- עובדים **לפי הסדר מההתחלה**, בעומק **תמצית + ניתוח**.
