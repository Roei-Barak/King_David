# מאגר פסוקי תהילים למחזמר — מחולק לקטגוריות

> **היקף:** מזמורים עם כותרת "לדוד" (כ-73) + מזמורים המיוחסים מסורתית לדוד.
> **מטרה:** חומר גולמי לכתיבת שירים — לא לשכתב, רק לבחור ולשלב.
> **סימנים:** 🎭 = הצעת תפקיד בשיר | 💡 = הערה דרמטית

---

## קטגוריה א' — I WANT / חלום הנוער
*תבנית מחזמר: I Want Song — הדמות מגלה לקהל מה היא רוצה מהחיים. בדרך כלל בפרולוג או בתחילת מערכה א'.*

| פסוק | מזמור | טקסט |
|------|-------|-------|
| כז, ד | כז | **"אַחַת שָׁאַלְתִּי מֵאֵת ה' אוֹתָהּ אֲבַקֵּשׁ — שִׁבְתִּי בְּבֵית ה' כָּל יְמֵי חַיַּי לַחֲזוֹת בְּנֹעַם ה' וּלְבַקֵּר בְּהֵיכָלוֹ"** |
| כז, י | כז | **"כִּי אָבִי וְאִמִּי עֲזָבוּנִי — וַה' יַאַסְפֵנִי"** |
| סט, ט | סט | **"מוּזָר הָיִיתִי לְאֶחָי — וְנָכְרִי לִבְנֵי אִמִּי"** |
| ח, ב | ח | **"ה' אֲדֹנֵינוּ מָה אַדִּיר שִׁמְךָ בְּכָל הָאָרֶץ"** |
| ח, ד-ה | ח | **"כִּי אֶרְאֶה שָׁמֶיךָ מַעֲשֵׂי אֶצְבְּעֹתֶיךָ — יָרֵחַ וְכוֹכָבִים אֲשֶׁר כּוֹנָנְתָּה. מָה אֱנוֹשׁ כִּי תִזְכְּרֶנּוּ — וּבֶן אָדָם כִּי תִפְקְדֶנּוּ?"** |
| כג, א-ב | כג | **"ה' רֹעִי לֹא אֶחְסָר — בִּנְאוֹת דֶּשֶׁא יַרְבִּיצֵנִי עַל מֵי מְנֻחוֹת יְנַהֲלֵנִי"** |
| קלא, א | קלא | **"ה' לֹא גָבַהּ לִבִּי וְלֹא רָמוּ עֵינַי — וְלֹא הִלַּכְתִּי בִּגְדֹלוֹת וּבְנִפְלָאוֹת מִמֶּנִּי"** |
| כב, יא | כב | **"עָלֶיךָ הָשְׁלַכְתִּי מֵרָחֶם — מִבֶּטֶן אִמִּי אֵלִי אָתָּה"** |
| קלט, יד | קלט | **"אוֹדְךָ עַל כִּי נוֹרָאוֹת נִפְלֵיתִי — נִפְלָאִים מַעֲשֶׂיךָ וְנַפְשִׁי יֹדַעַת מְאֹד"** |
| קמד, ג | קמד | **"ה' מָה אָדָם וַתֵּדָעֵהוּ — בֶּן אֱנוֹשׁ וַתְּחַשְּׁבֵהוּ"** |

🎭 **לשיר הפתיחה:** כז,ד = הפזמון ("אחת שאלתי"). כז,י + סט,ט = הבית של דוד (מרמז על הבדידות/יחס האב). ח,ד = הנער מביט בכוכבים.
💡 "אבי ואמי עזבוני" + "מוזר הייתי לאחי" הם הזוג המושלם לפסיכולוגיה של דוד הנסתר בין האחים.

## קטגוריה ב' — אהבה ויחסים (יונתן, מיכל, אבישג)
*תבנית מחזמר: Charm Song / Love Duet — שיר שמסביר מה דמות אחת מרגישה כלפי אחרת.*

| פסוק | מזמור | טקסט |
|------|-------|-------|
| קלג, א | קלג | **"הִנֵּה מַה טּוֹב וּמַה נָּעִים שֶׁבֶת אַחִים גַּם יָחַד"** |
| טז, ג | טז | **"לִקְדוֹשִׁים אֲשֶׁר בָּאָרֶץ הֵמָּה — וְאַדִּירֵי כָּל חֶפְצִי בָם"** |
| טז, ה-ו | טז | **"ה' מְנָת חֶלְקִי וְכוֹסִי — אַתָּה תּוֹמִיךְ גּוֹרָלִי. חֲבָלִים נָפְלוּ לִי בַּנְּעִמִים"** |
| קמה, ג | קמה | **"גָּדוֹל ה' וּמְהֻלָּל מְאֹד — וְלִגְדֻלָּתוֹ אֵין חֵקֶר"** |
| כב, כה | כב | **"כִּי לֹא בָזָה וְלֹא שִׁקַּץ עֱנוּת עָנִי — וְלֹא הִסְתִּיר פָּנָיו מִמֶּנּוּ"** |
| מה, ב | מה | **"רָחַשׁ לִבִּי דָּבָר טוֹב — אֹמֵר אֲנִי מַעֲשַׂי לְמֶלֶךְ"** *(מזמור חתונה)* |
| מה, ח | מה | **"אָהַבְתָּ צֶּדֶק וַתִּשְׂנָא רֶשַׁע — עַל כֵּן מְשָׁחֲךָ אֱלֹהִים אֱלֹהֶיךָ שֶׁמֶן שָׂשׂוֹן מֵחֲבֵרֶיךָ"** |
| קכב, א | קכב | **"שָׂמַחְתִּי בְּאֹמְרִים לִי — בֵּית ה' נֵלֵךְ"** |

🎭 **ליונתן ודוד:** קלג,א = הפתיחה הטבעית לדואט. טז,ה-ו = "חלקי נפל יפה" — ניתן לפרש גם על ידידות. מה,ב = פזמון.
🎭 **למיכל:** מה (המזמור כולו) — מזמור חתונה מלכותי.
💡 קלג שיש לו לחנים ידועים רבים — מתאים לשיר קל וקצבי.

## קטגוריה ג' — נרדף / פחד / בגידה (שאול, אחיתופל, אבשלום)
*תבנית מחזמר: Eleven O'Clock Number / Dark Moment — השיר שבו הכל מתמוטט. לרוב קרוב לסוף מערכה ב'.*

| פסוק | מזמור | טקסט |
|------|-------|-------|
| נה, יג-טו | נה | **"כִּי לֹא אוֹיֵב יְחָרְפֵנִי... וְאַתָּה אֱנוֹשׁ כְּעֶרְכִּי אַלּוּפִי וּמְיֻדָּעִי. אֲשֶׁר יַחְדָּו נַמְתִּיק סוֹד — בְּבֵית אֱלֹהִים נְהַלֵּךְ בְּרָגֶשׁ"** |
| נה, ז | נה | **"וָאֹמַר מִי יִתֶּן לִי אֵבֶר כַּיּוֹנָה — אָעוּפָה וְאֶשְׁכֹּנָה"** |
| נה, כג | נה | **"וְאַתָּה אֱלֹהִים תּוֹרִידֵם לִבְאֵר שַׁחַת — אַנְשֵׁי דָמִים וּמִרְמָה לֹא יֶחֱצוּ יְמֵיהֶם"** |
| נז, ב | נז | **"חָנֵּנִי אֱלֹהִים חָנֵּנִי כִּי בְךָ חָסָיָה נַפְשִׁי — וּבְצֵל כְּנָפֶיךָ אֶחְסֶה עַד יַעֲבֹר הַוּוֹת"** |
| נט, ב-ג | נט | **"הַצִּילֵנִי מֵאֹיְבַי אֱלֹהָי... פְּדֵנִי מִפֹּעֲלֵי אָוֶן"** |
| ג, ב | ג | **"ה' מָה רַבּוּ צָרָי — רַבִּים קָמִים עָלָי"** *(כותרת: "בברחו מפני אבשלום")* |
| ג, ד | ג | **"וְאַתָּה ה' מָגֵן בַּעֲדִי — כְּבוֹדִי וּמֵרִים רֹאשִׁי"** |
| לד, יט | לד | **"רַבּוֹת רָעוֹת צַדִּיק — וּמִכֻּלָּם יַצִּילֶנּוּ ה'"** |
| ז, ב | ז | **"ה' אֱלֹהַי בְּךָ חָסִיתִי — הוֹשִׁיעֵנִי מִכָּל רֹדְפַי וְהַצִּילֵנִי"** |
| נד, ה | נד | **"כִּי זָרִים קָמוּ עָלַי וְעָרִיצִים בִּקְשׁוּ נַפְשִׁי"** |
| נב, ד-ה | נב | **"רָעָה תַּחְשֹׁב לְשׁוֹנֶךָ — כְּתַעַר מְלֻטָּשׁ עֹשֵׂה רְמִיָּה. אָהַבְתָּ רָע מִטּוֹב — שֶׁקֶר מִדַּבֵּר צֶדֶק"** |
| קמב, ב-ג | קמב | **"אֶשְׁפֹּךְ לְפָנָיו שִׂיחִי — צָרָתִי לְפָנָיו אַגִּיד"** |
| כב, ב | כב | **"אֵלִי אֵלִי לָמָה עֲזַבְתָּנִי — רָחוֹק מִישׁוּעָתִי דִּבְרֵי שַׁאֲגָתִי"** |

🎭 **לסצנת הבריחה מאבשלום:** ג (כל המזמור) — כתרי "בברחו מפני אבשלום בנו."
🎭 **לאחיתופל:** נה,יג-טו — הפסוק המדויק על בגידת ידיד.
🎭 **לסצנת העין גדי / המערה:** נז,ב — "בצל כנפיך אחסה."
💡 "מי יתן לי אבר כיונה" (נה,ז) = שיר בריחה — רגע בו הגיבור רוצה לברוח מהכל. תבנית מחזמרית קלאסית.

## קטגוריה ד' — חטא ותשובה (בת-שבע, "אתה האיש")
*תבנית מחזמר: Reprise / Confession Song — שיר שמהפך שיר קודם, או שיר שבו הדמות מבינה את מעשיה.*

| פסוק | מזמור | טקסט |
|------|-------|-------|
| נא, ג | נא | **"חָנֵּנִי אֱלֹהִים כְּחַסְדֶּךָ — כְּרֹב רַחֲמֶיךָ מְחֵה פְשָׁעָי"** |
| נא, ה | נא | **"כִּי פְשָׁעַי אֲנִי אֵדָע — וְחַטָּאתִי נֶגְדִּי תָמִיד"** |
| נא, ו | נא | **"לְךָ לְבַדְּךָ חָטָאתִי — וְהָרַע בְּעֵינֶיךָ עָשִׂיתִי"** |
| נא, ט | נא | **"תְּחַטְּאֵנִי בְאֵזוֹב וְאֶטְהָר — תְּכַבְּסֵנִי וּמִשֶּׁלֶג אַלְבִּין"** |
| נא, יב | נא | **"לֵב טָהוֹר בְּרָא לִי אֱלֹהִים — וְרוּחַ נָכוֹן חַדֵּשׁ בְּקִרְבִּי"** |
| נא, יג | נא | **"אַל תַּשְׁלִיכֵנִי מִלְּפָנֶיךָ — וְרוּחַ קָדְשְׁךָ אַל תִּקַּח מִמֶּנִּי"** |
| נא, יד | נא | **"הָשִׁיבָה לִּי שְׂשׂוֹן יִשְׁעֶךָ — וְרוּחַ נְדִיבָה תִסְמְכֵנִי"** |
| נא, יט | נא | **"זִבְחֵי אֱלֹהִים רוּחַ נִשְׁבָּרָה — לֵב נִשְׁבָּר וְנִדְכֶּה אֱלֹהִים לֹא תִבְזֶה"** |
| לב, ג-ד | לב | **"כִּי הֶחֱרַשְׁתִּי בָּלוּ עֲצָמָי... כִּי יוֹמָם וָלַיְלָה תִּכְבַּד עָלַי יָדֶךָ"** |
| לב, ה | לב | **"חַטָּאתִי אוֹדִיעֲךָ — וַעֲוֹנִי לֹא כִסִּיתִי... וְאַתָּה נָשָׂאתָ עֲוֹן חַטָּאתִי"** |
| לח, ה | לח | **"כִּי עֲוֹנֹתַי עָבְרוּ רֹאשִׁי — כְּמַשָּׂא כָבֵד יִכְבְּדוּ מִמֶּנִּי"** |
| לח, יט | לח | **"כִּי עֲוֹנִי אַגִּיד — אֶדְאַג מֵחַטָּאתִי"** |

🎭 **לשיר "חטאתי":** נא (כל המזמור) — המקור. אבל נא,יב ("לב טהור") = הפזמון האפשרי, ונא,יט ("לב נשבר ונדכה") = השורה האחרונה.
🎭 **כ-Reprise:** לב,ה — "חטאתי אודיעך" יכול לחזור בקודה, שונה מגרסת הפרולוג.
💡 נא אורגנית הוא שיר תשובה — אפשר לקחת אותו כמעט שלם.

## קטגוריה ה' — אבהות ואובדן (אמנון, אבשלום, "בני בני")
*תבנית מחזמר: Lament — שיר אבל שמאפשר לקהל להרגיש את הכאב של הדמות.*

| פסוק | מזמור | טקסט |
|------|-------|-------|
| לח, יב | לח | **"אֹהֲבַי וְרֵעַי מִנֶּגֶד נִגְעִי יַעֲמֹדוּ — וּקְרוֹבַי מֵרָחֹק עָמָדוּ"** |
| כב, ב | כב | **"אֵלִי אֵלִי לָמָה עֲזַבְתָּנִי"** |
| כב, ז | כב | **"וְאָנֹכִי תוֹלַעַת וְלֹא אִישׁ — חֶרְפַּת אָדָם וּבְזוּי עָם"** |
| לא, יג | לא | **"כִּי שָׁמַעְתִּי דִּבַּת רַבִּים — מָגוֹר מִסָּבִיב"** |
| סט, ב-ג | סט | **"הוֹשִׁיעֵנִי אֱלֹהִים כִּי בָאוּ מַיִם עַד נָפֶשׁ... בָּאתִי בְמַעֲמַקֵּי מַיִם וְשִׁבֹּלֶת שְׁטָפָתְנִי"** |
| נה, ה-ו | נה | **"לִבִּי יָחִיל בְּקִרְבִּי — וְאֵימוֹת מָוֶת נָפְלוּ עָלָי. יִרְאָה וָרַעַד יָבֹא בִי — וַתְּכַסֵּנִי פַּלָּצוּת"** |
| קמג, ד | קמג | **"וַתִּתְעַטֵּף עָלַי רוּחִי — בְּתוֹכִי יִשְׁתּוֹמֵם לִבִּי"** |
| לט, ה | לט | **"הוֹדִיעֵנִי ה' קִצִּי — וּמִדַּת יָמַי מַה הִיא אֵדָעָה מֶה חָדֵל אָנִי"** |
| קב, ח | קב | **"שָׁקַדְתִּי וָאֶהְיֶה כְּצִפּוֹר בּוֹדֵד עַל גָּג"** |

🎭 **לרגע מות אבשלום:** נה,ה-ו = מה דוד מרגיש כשמגיע הבשורה. לח,יב = "אוהבי ורעי... עמדו מרחוק."
🎭 **לסצנת "אחיתופל":** קב,ח = בודד כציפור על גג — תמונה ויזואלית חזקה.
💡 "אלי אלי למה עזבתני" (כב,ב) — ציטוט הישיר ביותר לכאב הנטישה; הד ל"מוזר הייתי לאחי."

## קטגוריה ו' — מלכות ואחריות (ירושלים, הברית, הצדק)
*תבנית מחזמר: Anthem / Coronation Number — שיר הקרנה, שיר שמגדיר ייעוד ציבורי.*

## הערות מתודולוגיות

1. **כל הפסוקים כאן הם מקוריים מהתנ"ך** — לא מומצאים. מראי מקום מלאים ניתנים.
2. **לאמת לפני שימוש:** ניסוח הפסוקים מובא מזיכרון; מומלץ לאמת מול תנ"ך מנוקד לפני הכנסה לטקסט סופי.
3. **לחנים קיימים:** חלק מהפסוקים כבר קיימים כשירים ידועים (לחן כהן, קל, נד וכו') — ניתן להשתמש בלחן הידוע או להזמין לחן מקורי.
4. **"לולא האמנתי" (כז,יג)** — בטקסט המקורי חסר סוף המשפט (נקודות בסוף). מסורתית מפרשים שדוד לא סיים כי היה ירא לומר מה יהיה אם לא יאמין. הרגע הזה — הנפסק — הוא אחד החזקים ביותר שניתן לבנות עליו שיר.
