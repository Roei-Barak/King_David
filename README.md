# מחזמר דוד המלך 🎵👑

מחזמר במה על סיפור חייו של **דוד המלך** — מנער רועה צאן ועד זקנתו — לפי המקורות היהודיים (תנ"ך + חז"ל), עם רפרנסים לעת החדשה.

## הרעיון

דוד כדמות מופת של **אמונה דרך ייסורים**: למרות כל מה שעבר עליו, המשיך להאמין — ומתוך חייו צמחו **מזמורי תהילים**. כל תחנה משמעותית בחייו מקושרת למזמור, ולמקבילה מבת זמננו.

- **שפה:** עברית + לשון מקראית
- **טון:** מרגש ומכבד
- **חיבור לעת החדשה:** מקבילות תמטיות, סגנון מוזיקלי עכשווי, מספר/מסגרת מודרנית, ודמויות מקבילות
- **פורמט וקהל:** טרם נקבע

## מבנה

- `concept/` — חזון, תזה, טון
- `sources/` — מקורות + מיפוי מזמורי תהילים לתחנות החיים
- `research/` — ציר חיי דוד (התחנות) ומחקר רקע
- `characters/` — דמויות
- `outline/` — מתווה מערכות וסצנות
- `scenes/` — טיוטות סצנות
- `songs/` — מפת שירים ומילים
- `modern-references/` — מאגר רפרנסים לעת החדשה
