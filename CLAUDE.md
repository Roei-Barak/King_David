# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## מהות הפרויקט

זהו פרויקט **כתיבה יוצרת**, לא פרויקט תוכנה. המטרה: מחזמר במה על סיפור חייו של דוד המלך.
אין כאן קוד, build או tests — ה"מקור" הוא מסמכי Markdown בעברית (מקורות, מתווה, דמויות, סצנות, מילות שירים).

## הקונספט המוביל (לא לסטות ממנו בלי אישור)

- **תזה מרכזית:** דוד כדמות מופת של **אמונה דרך ייסורים** — למרות כל מה שעבר, המשיך להאמין וחיבר את מזמורי תהילים.
- **יריעה:** כל חייו, מנער רועה צאן ועד זקנתו והמלכת שלמה.
- **חוט שני:** **תהילים** — לכל תחנה משמעותית בחיי דוד מוצמד מזמור שצומח מתוך הרגע הדרמטי. ראה `sources/tehillim-map.md`.
- **חיבור לעת החדשה:** משולבים כל האמצעים — מקבילות תמטיות, סגנון מוזיקלי עכשווי, מספר/מסגרת מודרנית, ודמויות מקבילות בנות זמננו. ראה `modern-references/references.md`.
- **טון:** מרגש ומכבד — דרמה רגשית רצינית עם יראת כבוד למקורות.

## שפה וכתיבה

- **עברית** לדיאלוגים ולשירים, עם שילוב **לשון מקראית/פסוקים** ברגעים הרלוונטיים (במיוחד בציטוט תהילים ובדברי דוד).
- הקבצים נכתבים מימין לשמאל בתוכן; שמות הקבצים והתיקיות באנגלית לנוחות git.
- בעת ציטוט מקור — לציין מראה מקום מדויק (ספר, פרק, פסוק).

## כללי עבודה עם מקורות (קריטי)

- המקורות הם **תנ"ך + חז"ל** (מדרשים, תלמוד, פרשנים) + פרשנויות ספציפיות שהמשתמש מספק.
- **אין להמציא מדרשים, פסוקים או מובאות.** כל קביעה "מהמקורות" חייבת להיות מאומתת מול חומר ב-`sources/` או מול מה שהמשתמש סיפק. אם לא בטוח — לסמן `[לאמת]` ולשאול.
- המשתמש מנהל מקורות ב-NotebookLM אך **אין גישה ישירה** אליו מהסביבה הזו. מקורות מגיעים דרך Google Drive, הדבקה בצ'אט, או קבצים שמתווספים ל-`sources/`.
