# בריף למלחין — "מזמור לדוד"

> **מסמך זה מיועד למלחין/ת.** מפרט לכל שיר: תפקיד דרמטי, ז'אנר, טמפו, מפתח רגשי, מלחן-עוגן (כהשראה בלבד — לא לכלול בהפקה בלי רישיון), ומידע ביצוע.

---

## עקרונות מוזיקליים כלליים

1. **הגשר הוא המוזיקה עצמה** — הקהל פוגש את דוד המלך דרך קו מוזיקלי ישראלי-עכשווי, לא דרך מספר מסביר.
2. **שתי שכבות:** 11 שירים על לחנים ישראליים מוכרים (עוגן היכרות) + 11 שירים מקוריים (מרחב יצירתי).
3. **אין ציטוט מוזיקלי** — הלחנים המוכרים הם נקודת מוצא קולית, לא עיבוד.
4. **עברית מקראית בשירים** — לא לתרגם, לא לשחזר. הפסוק בפה הדמות.
5. **כינור כחוט שזור** — מוטיב כינור (ניגון קצר) חוזר בנקודות מפתח: פרולוג → מזמור נא → "ממעמקים" → פינאלה.

## שיר 1 — "אַחַת שָׁאַלְתִּי" (שיר פתיחה / I WANT SONG)

| פרמטר | ערך |
|--------|-----|
| **מיקום** | פרולוג — פתיחת המחזה |
| **מבצע** | דוד (נער) + מקהלה |
| **אורך** | 5-7 דקות |
| **תפקיד דרמטי** | הצגת ה-I WANT של דוד: לא מלוכה — **שייכות** ("שבת בבית ה'"). הבעיה המרכזית (הזרות) נחשפת. |
| **ז'אנר** | בלדה-פופ ישראלית, בנייה הדרגתית. מתחיל בכינור סולו → מקהלה → שיא פורה |
| **טמפו** | ♩=60 (איטי, נשימתי) → ♩=84 בשיא |
| **מפתח רגשי** | כמיהה שקטה → כאב הזרות → הכרזת אמונה |
| **עוגן-השראה** | "כינור דוד" (שימי תבורי / אביהו מדינה) — טיפוס מלודי עממי-ישראלי |

| **הערה למלחין** | הנגינה בכינור ב-intro היא "כינורו של דוד" — מוטיב 4-5 תווים שיחזור לאורך המחזה. |
