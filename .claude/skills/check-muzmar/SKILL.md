## בדיקה 1 — שפה: עברית מקראית ולא מודרנית

### מילות אזהרה (בפה דמויות תנ"כיות — שגיאה):
```bash
# חפש ביטויים מודרניים בקבצי סצנות
grep -n "בסדר\|רגע\|אז \|בעצם\|כן כן\|נו \|ממש\|פשוט\|וואו\|אוקי\|טוב טוב\|יאללה\|וכו'\|וכדומה\|בגדול\|בקטן\|יש פה\|אין פה\|אני מרגיש" \
  /home/user/King_David/scenes/*.md
```

### בדיקה ידנית — דיאלוג תקין לעומת פסול:

| ❌ מודרני (פסול) | ✅ מקראי (תקין) |
|-----------------|----------------|
| "בסדר, אני מבין" | "כַּאֲשֶׁר דִּבַּרְתָּ — כֵּן אֶעֱשֶׂה" |
| "רגע, תחכה" | "עֲמֹד פֹּה עַד אֲשֶׁר" |
| "אז מה אתה אומר?" | "מַה הַדָּבָר אֲשֶׁר תְּדַבֵּר?" |
| "אני לא מבין" | "לֹא יָדַעְתִּי אֶת הַדָּבָר" |
| "זה לא הוגן" | "לֹא כֵן יֵעָשֶׂה" |

### דיאלוג מול הערות במאי:
- הערות במאי: *מותרת* עברית מודרנית — הן לבמאי, לא לדמות
- `*(מסתכל עליו)*` — בסדר
- **דָּוִד**: "מסתכל עליו" — ❌ זה לא דיאלוג זה הוראה

## בדיקה 2 — מקורות: כל ציטוט עם (ספר פרק, פסוק)

```bash
# מצא שורות עם ציטוטים (גרשיים) בלי סימון מקור
# פסוקים בדרך כלל מגיעים עם מבנה עברי ייחודי
grep -n "\".*\"" /home/user/King_David/scenes/$FILE | \
  grep -v "\*(שמ\|תה\|מל\|בר\|שמ\"\|דב\|דה\"\|יש\|יר\|עמ\|הו\|יח\|זכ\|מש\|אי\|רו\|נח\|אי)"
```

### פורמט מקור תקין:
```
"וַיִּשְׁמַע דָּוִד" *(שמ"א כג, כה)*
"לֹא יָדְעוּ" *(תהילים מד, כב)*
"הוּא הָאִישׁ" [לאמת — מדרש שמואל]
```

### מה דורש `[לאמת]`:
- מדרש שלא אומת מול מקור ספציפי
- פסוק שאתה לא בטוח בדיוקו
- "נאמר ש..." בלי מקור

### מה **לא** דורש סימון:
- הוראות במאי
- תיאורי עלילה פרי-דמיון (לא ציטוט)
- כותרות פנימיות

## בדיקה 3 — שלוש השפות: לא מעורבבות

### כלל:
```
דיאלוג → פרוזה (אסור חרוז)
מונולוג → שיר חופשי (אסור חרוז, אסור פרוזה)
שיר → חרוז קבוע (אסור פרוזה, אסור שיר חופשי)
```

### בדיקה ידנית — שאל על כל שכבה:

**דיאלוג — האם יש חרוז?**
```bash
# מצא שורות דיאלוג (מגרש תנ"כי בתחילה) שמסתיימות באותה הברה
grep -n "^\*\*[^*]*\*\*:" /home/user/King_David/scenes/$FILE | head -20
# אם שורות קצרות וחרוזיות — שגיאת סגנון
```

**מונולוג — האם יש פרוזה?**
- מונולוג תקין: שורות קצרות, אין פסיקים ארוכים, אין "כי... אשר... לכן..."
- מונולוג פסול: משפטים ארוכים, לוגיקה לינארית, הסבר

**שיר — האם יש חרוז עקבי?**
- בדוק שכל בית/פזמון שומר מטר דומה
- שורות שוות בערך באורכן
