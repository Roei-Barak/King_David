# מדריך כתיבה — מחזמר "מזמור לדוד"

## עקרונות יסוד (לא לחרוג בלי אישור מפורש)

**תזה:** דוד כדמות מופת של אמונה דרך ייסורים. חוט הזהב: "כל עצירת יד מולידה כינור."

**שלוש השפות — חוק ברזל:**
| שכבה | רגיסטר | דוגמה |
|------|---------|--------|
| דיאלוג | פרוזה מליצית מקראית | "מַה לִּי וְלָכֶם בְּנֵי צְרוּיָה" |
| מונולוג פנימי | שיר חופשי בסגנון תהילים | שורות קצרות, חזרה, עלייה וירידה |
| שיר מוזיקלי | חרוז קבוע + מטר | "הֲלָלוּיָהּ / עַל כָּל אֲשֶׁר הָיָה" |

**אסור לערבב** — מונולוג בחרוז, דיאלוג בשיר חופשי = שגיאה.

## מצב א׳ — כתיבת סצנה חדשה

### קבל מהמשתמש:
- פרק תנ"כי (כגון: שמ"א כג)
- דמויות מרכזיות
- מה רגש/נושא מרכזי

### בדוק לפני הכתיבה:
```bash
# וודא שהסצנה לא קיימת
ls /home/user/King_David/scenes/ | grep -i <שם>
# קרא את מפת המזמורים הרלוונטי
grep -A3 "פרק <מספר>" /home/user/King_David/sources/tehillim-map.md
# קרא את המדרשים הרלוונטיים
grep -A5 "<שם-דמות\|שם-אירוע>" /home/user/King_David/scenes/scene-midrashim.md
```

### מבנה סצנה:
```markdown
# כותרת — שם הסצנה
### מקור: שמ"א כג, טו-יח | ~X דקות

## חלק א': כותרת חלק
**שמואל א' כג, טו | ~X דקות**

*(הוראות במאי: בעברית מודרנית — מה קורה פיזית)*

**דָּוִד** *(תנועה)*:
"דִּבּוּר בְּפְּרוֹזָה מְלִיצִית..." *(שמ"א כג, יז)*

*(תגובה)*

## מזמור — [שם] *(מונולוג פנימי)*
*(שיר חופשי — כמו תהילים)*
שורה קצרה
עוד שורה
הרחבה — ואז...
שיא

## שיר: "כותרת שיר" *(אם יש)*
*(חרוז + מטר קבוע)*
שורה א — ריתמוס
שורה ב — חרוז
```

### כללים קריטיים:
- כל ציטוט מקראי: `*(שמ"א כג, יז)*` — מיד אחרי הציטוט
- מדרש לא מאומת: לסמן `[לאמת]`
- **אסור להמציא פסוקים** — אם לא בטוח, [לאמת]
- קול ייחודי לכל דמות (ראה `characters/` לפירוט)

## מצב ב׳ — Script Doctor (ליטוש סצנה קיימת)

### חוקי ברזל:
1. **לא למחוק** — לחדד בלבד
2. **לא לקצר** — להוסיף עומק
3. **לא לשנות מבנה** — לשפר מה שיש

### פעולות Script Doctor:
```bash
# קרא את הסצנה
cat /home/user/King_David/scenes/<שם>.md

# בדוק: האם יש ציטוטים ללא מקור?
grep -n "\"" <קובץ> | grep -v "\*(שמ\|תה\|מל\|דב\|בר\|בר"

# בדוק: האם יש עברית מודרנית בפה דמויות?
# מילים אסורות בדיאלוג: "בסדר", "רגע", "אז", "בעצם", "כן כן", "נו"
```

### סדר בדיקה:
1. **שפה** — האם כל דיאלוג בפרוזה מקראית? (ר׳ `/check-muzmar`)
2. **קולות** — האם כל דמות נשמעת אחרת?
3. **סאבטקסט** — האם יש מה שלא נאמר?
4. **שלוש שפות** — האם לא מעורבבות?
5. **מקורות** — ציטוט ללא `*(מקור)*`?
6. **חוט זהב** — היכן עצירת היד? מה הכינור שצמח?

### לכל שיפור — להסביר *למה*:
```markdown
*(שיפור: הפסוק המקורי "לא ידעתי" הוחלף ב"מה לי ולכם" כי רגיסטר מקראי)*
```

## מצב ג׳ — כתיבת שיר

### קבל מהמשתמש:
- תפקיד דרמטי (I WANT / לאמנט / שיר ניצחון / דואט...)
- דמות שרה
- פסוקי תהילים רלוונטיים (מ-`sources/tehillim-map.md`)
- לחן/ז׳אנר אם ידוע

### בדוק קודם:
```bash
cat /home/user/King_David/songs/song-map.md
grep -A5 "<שם-שיר\|תמונה>" /home/user/King_David/songs/song-map.md
```

### מבנה שיר (חרוז):
```
## בית א׳
[4-6 שורות, מטר קבוע, חרוז ABAB או AABB]

## פזמון
[4 שורות, חוזר — פשוט וחזק]
[ניתן לזמר מהר]

## בית ב׳
[אותו מבנה — תוכן מתקדם]

## פזמון

## שיא (bridge)
[שינוי מטר — עוצמה]

## פזמון (גרסה שינוי קל)
```

### דרישות שיר:
- לפחות פסוק תהילים אחד משולב (במקור, לא תרגום)
- מטרה דרמטית: מה השיר מקדם בעלילה?
- לסמן בגוף הקובץ: `**[תפקיד דרמטי: ...]**`
