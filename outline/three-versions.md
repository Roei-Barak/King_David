# שלוש גרסאות — מה נכנס ומה יוצא

> כל גרסה היא מחזה שלם — אחרת דגשים, אחרת זמן, אחרת תמונת דוד.
> הבסיס הקבוע (קיים לכולן): משיחה → גוליית → יהונתן → בריחה → מערות → מות שאול → ירושלים → ארון → נבואת נתן → מפיבושת → בת-שבע → "אתה האיש" → אמנון ותמר → מרד אבשלום → מות אבשלום → אפילוג

---

## גרסה א' — "נעים זמרות ישראל"
### הדמות: דוד כמשורר-נביא שחיי הרוח שלו הם הסיפור האמיתי

**תזה:** לא המלחמות עשו את דוד — השיר עשה אותו.
כל מה שעבר עליו — הפך לתהילים. זה הניצחון האמיתי.

**מבנה:** כינור כחוט המחבר. כל תמונה פותחת או נסגרת בדוד יוצר שיר.

**מה נכנס מהחומר החדש:**

| סצנה | למה כן |
|------|---------|
| [מ-א] כינור בחצות | **ליבה** — זה הדמות. בכל לילה, בכל מצב — קם וכותב |
| [מ-ז] "לא תחפוץ בזבח" | **קריטי** — התפילה כקרבן = כל תהילים ניב מזה |
| [מ-ו] "שגיאות מי יבין" | **קריטי** — עומק התשובה האמיתית |
| [מ-יב] "אהבה שאינה תלויה" | כן — יהונתן כמוזה הראשונה |
| [מ-יא] ישי לא האמין | כן — "מוזר הייתי לאחי" = שורש |
| [מ-ה] הרועה החכם | כן — מקור הרגישות |
| [מ-ח] שבת במדבר | כן — שיר גם בתחתית |
| scene-chr25-singers.md — ארגון משוררים | **קריטי** — כאן מבין שהמורשת שירית |
| scene-22-23-david-song.md — "נעים זמרות ישראל" | **קריטי** — הסיום |

**מה יוצא / מוקצר:**

| סצנה | למה לא / לקצר |
|------|----------------|
| מלחמת עמון (scene-10b) | פחות רלוונטי — מונטאז' של שורה |
| שיבת אבשלום-יואב (scene-13e) | מקוצר — שתי שורות |
| רצפה בת איה (scene-13g) | אופציונלי — יפה אבל עצור את הקצב |
| פגישות שמעי ומפיבושת בשיבה (scene-13f) | מקוצר |
| גת/המשוגע (04b) | אופציונלי |

**שירים מרכזיים:** "מוזר הייתי לאחי" → "ממעמקים" → "חטאתי" → "הללויה" → "נעים זמרות ישראל"
**טון:** ליריקה. אינטימי. קרוב. כאילו אנחנו בחדר עם האיש שכותב.
**זמן:** ~3 שעות

## גרסה ב' — "הצדיק הנרדף"
### הדמות: דוד כאיש שכולם נגדו — ובוחר שוב ושוב לא להחזיר רע

**תזה:** דוד לא גיבור כי ניצח — גיבור כי *לא* הרג כשיכול.
(זו ההקבלה לארי פוטר — הכוח שבסירוב להשתמש בכוח)

**מבנה:** שלוש "נסיונות הריגה" כציר: שאול בעין גדי ← נבל ← שמעי. בכל פעם — ידו נעצרת. זה גדל.

**מה נכנס מהחומר החדש:**

| סצנה | למה כן |
|------|---------|
| [מ-יג] הכינור אומר "לא אתה — הוא" | **פתיחה** — הסוד שכולם יודעים חוץ מדוד |
| [מ-יא] ישי לא האמין | כן — "כולם לא האמינו בו" |
| [ד-ב] גת/המשוגע | **קריטי** — מבזה עצמו כדי לשרוד, לא נלחם |
| [ד-ג] טבח נוב — "אנכי סבותי" | **קריטי** — האשמה הראשונה. הוא גרם, לא ביודעין |
| [ד-ד] פגישה אחרונה עם יהונתן | כן — הברית שחייה |
| [ד-ה] נבל ואביגיל | **קריטי** — עוצר עצמו בזכות אישה |
| scene-05b — שאול ובעלת האוב | כן — הצד השני של "הצדיק הנרדף" |
| scene-13f — שמעי + פרידת ברזילי | **קריטי** — "הניחו לו ויקלל" = שיא |
| [מ-י] "אפול ביד ה'" | כן — גם בעונש, בוחר ה' |
| scene-15 — "ממך הכל" | **קריטי** — הסיכום: לא ניצח, מסר |

**מה יוצא / מוקצר:**

| סצנה | למה לא |
|------|--------|
| ארגון משוררים | מקוצר — לא מרכזי |
| [מ-ח] שבת במדבר | אופציונלי |
| [מ-ט] בית המרחץ | יוצא — לא מתאים לטון |
| רצפה בת איה | אופציונלי — יפה אבל עצור |

**שירים מרכזיים:** "גם כי אלך" (גוליית) → "גרשוני היום" (עין גדי) → "ממעמקים" (מרד) → "אפול ביד ה'" → "הללויה"
**טון:** דרמה. מתח. שאלת המוסר — כל סצנה היא בחירה.
**זמן:** ~3.5 שעות

## גרסה ג' — "אב ובנים"
### הדמות: דוד כאב — כישלון האהבה הגדול ביותר שלו

**תזה:** דוד ניצח בכל — ובביתו: נכשל.
שלושה "בנים": יהונתן (בן-נפש שנלקח), אבשלום (בן שמרד), שלמה (בן שלא הכיר).
זה מחזה על אב שלא ידע לפתוח דלת.

**מבנה:** ציר בנים — כל מערכה מוקדשת לבן אחר.

**מה נכנס מהחומר החדש:**

| סצנה | למה כן |
|------|---------|
| [מ-יא] ישי לא האמין | **פתיחה** — הכל מתחיל כאן. דוד שלא קיבל אהבת אב |
| [ד-ד] פגישה אחרונה עם יהונתן | **קריטי** — "הבן" הרוחני, האהבה שכן עבדה |
| scene-13e — שיבת אבשלום ("וישק המלך") | **קריטי** — הנשיקה המאוחרת מדי |
| [ד-יד] — "לאט לי לנער" | כבר קיים — **קריטי** |
| scene-13f — ברזילי הגלעדי | **קריטי** — מראה ל"דוד כאב": יצא מה שיצא |
| scene-13g — רצפה בת איה | **קריטי** — אמא שלא עזבה = אנטיתזה לדוד כאב |
| scene-15 — "חזק והיה לאיש" (צוואה לשלמה) | **קריטי** — פעם ראשונה שאומר לבנו מה שצריך |
| [מ-א] כינור בחצות | כן — מה דוד עשה ב"בדידות" |
| [מ-ב] אחיתופל רבו | כן — "אב רוחני" שגם הוא בגד |

**מה יוצא / מוקצר:**

| סצנה | למה לא |
|------|--------|
| מלחמות רבות | רקע בלבד |
| שאול ובעלת האוב | מקוצר — אחר |
| [מ-ג] שפיכת המים | אופציונלי |
| ארגון משוררים | יוצא |
| גת/המשוגע | מקוצר |

**שירים מרכזיים:** "ברית אחים" (יהונתן) → "חצי הפרדה" → "לא אנכי — בני" (שלמה) → "בני אבשלום" → "חזק והיה לאיש"
**טון:** אינטימי. קרוב. רגשי. פחות מלחמות — יותר חדרים קטנים ושיחות שלא היו.
**זמן:** ~2.5 שעות (קצר יותר)

## השוואה מרוכזת

| קריטריון | גרסה א' — משורר | גרסה ב' — נרדף | גרסה ג' — אב |
|-----------|-----------------|-----------------|--------------|
| **שאלת מפתח** | מה הופך ייסורים לשיר? | מה הופך ריסון לגדולה? | מה הופך אב לכישלון? |
| **שיא** | "ממך הכל ומידך" | "הניחו לו ויקלל" + "אפול ביד ה'" | נשיקת שלמה + "חזק והיה לאיש" |
| **מדרש מרכזי** | כינור בחצות | גת/המשוגע | ישי לא האמין |
| **הדמות שעולה** | דוד לבד עם ה' | דוד מול אויבים | דוד מול ילדיו |
| **הקבלה מודרנית** | לאונרד כהן, מה שמאחורי הלחן | ארי פוטר / גנדי | כל אב ובנו |
| **זמן** | ~3 שעות | ~3.5 שעות | ~2.5 שעות |
| **קהל יעד** | ספרותי, מוזיקלי | רחב, דרמטי | אישי, רגשי |

## מה שמשותף לכל שלוש (הבסיס שלא נסחר)

אלה הסצנות שחייבות להיות בכל גרסה — הן הנשמה:

1. **המשיחה** — "לא אשר יראה האדם"
2. **גוליית** — אומץ ואמונה
3. **עין גדי** — "חלילה לי מה'"
4. **נבואת נתן** — "לא אתה תבנה"
5. **"אתה האיש"** — חטא ותשובה
6. **"בני אבשלום"** — שבר אבהי
7. **"ממך הכל"** — הסיכום

## המלצתי

**לקחת את גרסה ב' כשלד ולהוסיף לה את ליבת גרסה ג'.**

כלומר: הציר של "הצדיק הנרדף" (שהוא גם הקבלת הארי פוטר שדיברנו עליה) — עם החוט הפנימי של **"האב שלא ידע לפתוח דלת"** כשכבה שנייה. זה נותן:
- דרמה חיצונית חזקה (גרסה ב')
- עומק רגשי אישי (גרסה ג')
- חוט התהילים (גרסה א', קיים ממילא)

**זמן: ~3.5 שעות — 15 תמונות + הפסקה.**
